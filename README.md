# Gyou-dori

Gyou-dori >>> https://www.w3.org/TR/jlreq/#processing_of_gyoudori

## Usage

\gyoudori[#1]{#2}{#3}

## Arguments (option): #1

The heading text postions.

- t: top in the area
- c: center in the area
- b: bottom in the area

## Arguments: #2

The line numbers of the area.

## Arguments: #3

The heading text.
